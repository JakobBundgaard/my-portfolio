Projekt Beskrivelse

ChatBot er et projekt fra PBA i Webudvikling, hvor vi skulle bygge en simpel chatGPTclon.

Backenden er bygget i PHP og frontenden er bygget i HTML, CSS og JavaScript.

Projektet har kørt i 3 iterationer.

I 1. iteration skulle der bygges en serverside version.

I 2. iteration skulle serverside versionen konverteres til en asynkron clientside version med JS fetch API.

I 3. iteriation skulle der arbejdes kollaborativt i grupper med Github som version control,

og opgaven var at koble en mySQL database på projektet, så chat historikken blev gemt i en tabel.

3. Iteration

Her er vores SQL queries til at lave tabellerne.

I responses tabellen har vi et autoincrementet responses_id, som er primary key og to

varchar attributter til spørgsmål og svar.

I chat_history tabellen har vi en primary key history_id, to varchar attributter

user_messages og response, til brugerens input og chatbottens svar.

Så har vi chat_id, som er foreign key, en reference til responses tabellen.

Users tabellen er til så, der på sigt, kan oprettes flere brugere som kan

se deres egen chat historik

Her forbinder vi til databasen med mysqli_connect.

først includer vi sql.php, hvor database informationen ligger.

De ligge i en seperat fil, som er gitignored, så login info

ikke bliver commited til Github
